# Local Game Hub

A local-first game portal built with React + TypeScript + Vite + Tailwind + Zustand.

## Features
- Home, Games, Players, Leaderboard, Settings pages
- Tic Tac Toe (PvP + Bot), Memory Card, Number Puzzle
- Local leaderboard and game history
- localStorage persistence
- Dark mode
- Export/import local JSON data
- Reset local data

## Run
```bash
npm install
npm run dev
```
